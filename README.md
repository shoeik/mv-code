# mv-code

Static interaction prototypes.

## Preview

This repository is designed to be published with GitHub Pages from the root of the `main` branch.

- Index: `https://shoeik.github.io/mv-code/`
- Works: open each item from the index page.

## Structure

```txt
works/
  <work-name>/
    analysis.md
    codepen.md
    source.mp4
    implementation/
      index.html
      style.css
      script.js
```

Each implementation is plain HTML/CSS/JS and can run without a build step.
